# 시퀀스
  * 문자열 : 문자들의 나열
    * 모든 문자는 str 타입
    * 문자열은 작은 따옴표나 큰 따옴표를 활용
    * 인덱싱
      * 인덱스를 통해 특정 값에 접근할 수 있음
      > * 0부터 시작한다!!
    * 결합  
      * 'hi'+ 'hello'
    * 반복 
      * 'hi!'*
    * 포함 
      * 'a' in 'apple'
    * 문자열 내에서 특정 문자나 조작을 위해서 역슬래시('\\')를 활용 하여 구분
      * \'안녕'\
      * \n 엔터, \t 탭
    
  * 인덱싱
    * 인덱스를 통해 특정 값에 접근할 수 있음
      * s[1] = 'b'
  * 슬라이싱
    * s[2:5]
  * 문자열 슬라이싱
    * s[2:5] = 'cde'
